About MockU

MockU is a generic university template for many commuinty-source web applications. Template originally created by The University of Nebraska–Lincoln for inclusion in various community source projects:

  • MediaHub A video and audio aggregator and distributor

General Requirements

This template uses a few standard and popular technologies:

  • jQuery
Other Technologies

While used in this template, the following are not techincally required:

  • HTML5
  • CSS3
  • LESS CSS
